# CS 300 - Data Structures and Algorithms

## Overview

This repository contains projects completed for **CS 300: Data Structures and Algorithms** at Southern New Hampshire University.

Throughout this course, I learned how different data structures and algorithms affect software performance, efficiency, and maintainability. I designed solutions using vectors, hash tables, and binary search trees before implementing a complete academic advising application in C++.

## Repository Structure

### Project One
Project One focuses on designing and evaluating data structures through algorithm design, pseudocode, and runtime analysis.

### Project Two
Project Two implements a C++ academic advising application using a Binary Search Tree to efficiently store, search, and display course information.

- Load course data from a CSV file
- Print all courses in alphanumeric order
- Search for individual courses
- Display course prerequisite information
