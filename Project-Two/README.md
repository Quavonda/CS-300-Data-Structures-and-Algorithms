# Project Two - Academic Advising Program

## Project Summary

This project demonstrates the use of a Binary Search Tree to create an academic advising application. The program reads course information from a CSV file and allows users to:

- Load course data
- Print all courses in alphanumeric order
- Search for a specific course
- Display prerequisite course information

## Skills Demonstrated

- C++
- Binary Search Trees
- File Input
- Object-Oriented Programming
- Searching Algorithms
